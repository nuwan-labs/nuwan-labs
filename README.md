# Nuwan Medawaththa

Junior Bioinformatics engineer at a national research institute. MSc in Computer Science at the University of Colombo School of Computing. I build AI-powered tools, full-stack applications, and knowledge systems.

When I see a manual process, I want to make it disappear. When I see scattered knowledge, I want to make it searchable. When I see someone struggling to learn, I want to build something that helps.

## What I Work With

**Languages:** Python, TypeScript, JavaScript, SQL, Bash, R, Nextflow

**AI:** LLM API integration, multi-agent orchestration, RAG pipelines, streaming responses, vision AI, PyTorch, HuggingFace, PEFT/LoRA

**Frontend:** React, Next.js, Astro, Tailwind CSS, HTML5 Canvas, Three.js/WebGL

**Backend:** FastAPI, Django/DRF, Node.js, PostgreSQL, SQLite, Firebase

**Infrastructure:** Docker, Singularity, AWS, Google Cloud, Vercel, Linux, CI/CD

## What I'm Building

🔬 **Domain knowledge extraction** — Turning curated research papers into structured knowledge graphs that sit alongside scientific pipelines and support decision-making at critical nodes

🧪 **AI-powered tools for real problems** — Tender evaluation with RAG, research orchestration with multi-agent collaboration, tutoring with adaptive mastery tracking

🌿 **Knowledge systems** — Making scattered domain expertise searchable and queryable, from medicinal plants to phytochemical databases
